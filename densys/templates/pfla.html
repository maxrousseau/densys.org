PFLA: Python Facial Landmark Analysis

  • Author: Maxime Rousseau
  • License: MIT
  • Version: 1.0.0
  • Source: Github

Description

This project came to be due to a need for automating the image annotation process to be able to compare the faces of various subjects samples and compare them. Traditionally such project would require training of the multiple scientists to annotate images and use kappa scores to ensure that these annotations were valid amongst the investigators. This is quite cumbersome, prone to bias and slow for large datasets. Hence, the need for automation of this process.

Densys makes use of pre-trained machine learning models to accomplish this task of annotating faces with landmarks. There are two model implementations required to do so: a face detector and a face annotator. For facial detection we used MTCNN (Multi-Task Convolutional Neural Network) and for annotation FAN (Face Alignment Network).

The program extracts the face bounding box, 68 landmarks and computes facial metrics for a single image or a folder containing multiple images (i.e. a sample/group). It outputs the results to a file of the user's choice (csv, xlsx, hdf5, pkl). This allows to conduct statistical analysis with his platform of choice (Excel, R, Matlab, Python, etc.)
